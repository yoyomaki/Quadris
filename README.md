Quadris
=======

Developers: Qian Qiao, Yunhao Xue  
Date: 07/30/2014

======

Quadris is a Latinsation of the game Tetris.   

======

Command:

======

* left
* right
* down
* clockwise
* counterclockwise
* drop
* levelup
* leveldown
* restart
* EOF terminates the game
* example: accpet lef to distinguish left from levelup
* prefix: 3ri means move to the right three times
